B — Panel dual: Sugerencias + Mi Pedido

Separación clara entre lo que el sistema recomienda (izquierda) y lo que el usuario está construyendo (derecha). El usuario arrastra o hace clic para mover items al pedido.

C — Asistente paso a paso (wizard)

Paso 1: selecciona productos. Paso 2: ajusta cantidades y precios. Paso 3: revisa el resumen y envía. Excelente para usuarios que se confunden con la pantalla completa.

⏱ Complejidad: media · Como SAP / Alegra Compras

💡 Mi recomendación: B — panel dual. Separa claramente "lo que el sistema ve" de "lo que yo estoy pidiendo". Más potente que A sin la rigidez del wizard de C.
